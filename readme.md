# ⏱️ Cronômetro

## 🔗 Demo


## 📌 Descrição
Cronômetro com iniciar, pausar e resetar.

## 🚀 Funcionalidades
- Contagem em tempo real
- Formatação HH:MM:SS
- Botões de controle
- Animações no tempo

## 🧠 O que eu aprendi
- `setInterval` e `clearInterval`
- Controle de estado (`rodando`)
- Conversão de tempo (segundos → horas/minutos)
- Uso de `%` (módulo)
- Formatação com `padStart`

## 🧩 Desafios enfrentados
- Bug de múltiplos intervalos
- Formatação correta do tempo
- Controle de execução
